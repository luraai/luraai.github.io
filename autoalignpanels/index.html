AutoAlignPanels

One hotkey. Perfect window grid. Every time.

Instantly arrange all windows of the same app into a clean grid layout. No dragging, no snapping zones — just press a key.

Download on the Mac App Store

Features

■■
■■

Smart Grid Layout

Automatically calculates the optimal grid (2x2, 3x2, etc.) based on the number of windows. Fewer windows? Empty cells are left clean.

⌨

Customizable Hotkey

Default is Shift+Ctrl+P, but change it to any key combination you like via the built-in shortcut recorder.

⚙

Menu Bar App

Lives quietly in your menu bar. Click the icon to access settings, or hide it entirely and use only the hotkey.

⋮

Adjustable Gaps

Set the spacing between windows to your preference — from tight edge-to-edge to comfortable padding.

►

Same-App Focus

Only aligns windows of the currently active application, keeping your other workspaces untouched.

🚀

Apple Silicon Native

Built exclusively for Apple Silicon Macs. Lightning fast, minimal resource usage.

How It Works

1

Open multiple windows

Open several windows of the same app — Terminal, Finder, Safari, anything.

2

Press the hotkey

Hit Shift+Ctrl+P (or your custom shortcut) while the app is in focus.

3

Windows snap to grid

All visible windows of that app instantly arrange into a perfect grid layout.
