자주 묻는 질문

계정을 만들어야 하나요?

아니요. kWord는 계정이 없습니다. 설치하면 바로 학습을 시작할 수 있습니다.

기기를 바꾸면 학습 기록이 사라지나요?

같은 Apple 계정으로 iCloud 동기화를 켜두셨다면 새 기기에서 이어집니다. 동기화를 쓰지 않으시는 경우, 설정 → 데이터 내보내기로 파일을 만들어 새 기기에서 가져오기 하시면 됩니다.

구독은 어떻게 해지하나요?

iOS 설정 → Apple 계정 → 구독에서 kWord를 선택해 해지할 수 있습니다. 해지는 Apple을 통해서만 가능하며, 저희가 대신 해지해 드릴 수는 없습니다.

환불을 받고 싶습니다.

App Store 결제의 환불은 Apple이 처리합니다. reportaproblem.apple.com 에서 요청해 주세요.

구매한 콘텐츠가 보이지 않습니다.

앱 안의 설정 → 구매 복원을 눌러주세요. 그래도 해결되지 않으면 구매하신 Apple 계정으로 로그인되어 있는지 확인 후 연락 주세요.

발음 평가가 동작하지 않습니다.

이 기능은 기기 안에서만 음성을 인식합니다. iOS 설정에서 kWord의 마이크·음성 인식 권한이 허용되어 있는지 확인해 주세요. 기기가 온디바이스 음성 인식을 지원하지 않으면 기능이 비활성화됩니다.

단어나 예문에 오류가 있습니다.

알려주시면 고치겠습니다. 앱의 설정 → 의견 보내기 또는 위 이메일로 해당 단어와 함께 보내주세요.

내 학습 기록이 다른 사람에게 보이나요?

아니요. 리그(순위표)에 직접 참여를 켠 경우에만 자동 생성된 익명 이름과 주간 XP가 순위표에 표시됩니다. 학습한 단어·오답·녹음은 어떤 경우에도 공개되지 않습니다. 자세한 내용은 개인정보처리방침을 참고하세요.
