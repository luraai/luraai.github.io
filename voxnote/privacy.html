개인정보처리방침

최종 수정일: 2026년 7월 13일 · English

VoxNote(이하 "본 앱")는 회의를 녹음하고, 전사하고, 요약하는 앱입니다. 이 세 가지가 모두 사용자의 기기 안에서 이루어집니다.

핵심 요약. 음성 파일과 전사문은 기본적으로 기기를 떠나지 않습니다. 운영자는 사용자의 회의 내용을 수신하지도, 저장하지도, 열람하지도 않습니다. 인터넷을 꺼도 앱의 모든 기능이 그대로 동작합니다.

1. 운영자가 수집하는 개인정보 — 없음

본 앱은 사용자의 개인정보를 일절 수집하지 않습니다.

  • 운영자는 사용자의 음성·전사문·요약을 수신하거나 보관하는 서버를 운영하지 않습니다.
  • 앱에 분석·마케팅 SDK가 포함되어 있지 않으며, 사용 기록을 수집·분석하지 않습니다.
  • 기기 식별자나 광고 식별자를 수집하지 않고, 다른 회사·앱·웹사이트의 데이터와 연결하지 않습니다.
  • 본 앱에는 광고가 없으며 추적 기술을 사용하지 않습니다.
  • 계정 가입이 필요 없습니다. 이메일이나 이름을 요구하지 않습니다.

2. 앱이 기기에 저장하는 정보

아래 정보는 사용자가 만든 것으로, 사용자의 기기에만 저장됩니다. 운영자는 이 데이터에 접근할 수 없으므로 Apple의 정의상 이는 "수집"에 해당하지 않습니다.

  • 녹음된 음성 파일
  • 전사문(받아쓰기 결과)과 화자 구분 정보
  • AI가 생성한 요약·회의록
  • 노트 제목, 폴더, 북마크, 하이라이트
  • 앱 설정(인식 언어, 사용할 AI 모델 등)

데이터베이스는 기기에 암호화되어 저장됩니다(ChaCha20-Poly1305).

3. AI 처리 — 기본은 온디바이스

음성 인식(STT)과 요약(LLM)은 기기에 내려받은 모델로 실행됩니다. 오디오도 전사문도 이 과정에서 밖으로 나가지 않습니다.

  • 음성 인식: whisper.cpp · SenseVoice (온디바이스)
  • 요약·문서 생성: llama.cpp (온디바이스)

AI 모델 파일은 최초 1회 다운로드가 필요하며, 이때 모델 배포처(예: Hugging Face)에 모델 파일 요청만 전송됩니다. 사용자의 음성이나 전사문은 전송되지 않습니다.

4. 외부 AI 사용 — 사용자가 켤 때만

사용자가 원한다면 OpenAI 등 외부 AI 서비스를 직접 선택해 요약에 사용할 수 있습니다. 이 기능은 기본적으로 꺼져 있습니다.

  • 외부 AI를 켜면, 앱은 무엇이 전송되는지 미리 알려줍니다(예: "이 노트의 전사 텍스트가 OpenAI 서버로 전송됩니다. 오디오 파일과 다른 노트는 전송되지 않습니다").
  • 앱 화면에서 초록색은 "이 기기 안에서 처리됨", 파란색은 "데이터가 기기를 떠날 수 있음"을 뜻합니다. 어떤 엔진이 도는지 언제나 확인할 수 있습니다.
  • 전송된 데이터는 해당 서비스 제공자의 개인정보처리방침에 따릅니다. 운영자는 그 데이터를 수신하거나 중계하지 않습니다 — 앱이 사용자의 API 키로 직접 호출합니다.
  • 외부 AI를 켜지 않으면 아무것도 전송되지 않습니다.

API 키는 운영자에게 전송되지 않으며, 운영체제의 보안 저장소(macOS 키체인)에 보관됩니다.

5. 기기 권한 (마이크)

본 앱은 마이크 권한을 회의 녹음 목적으로만 요청하며, 사용자가 녹음을 시작할 때만 사용합니다. 녹음된 음성은 위 2항과 같이 기기에만 저장됩니다.

6. 제3자 제공 및 공유 — 없음

운영자는 사용자의 데이터를 제3자에게 제공·판매·공유하지 않습니다. 애초에 운영자가 수신·보유하는 데이터가 없습니다.

7. 데이터 삭제

  • 노트는 앱 안에서 휴지통으로 옮기고, 휴지통에서 영구 삭제할 수 있습니다.
  • 음성 파일만 따로 삭제할 수도 있습니다(전사문과 요약은 유지됩니다).
  • 모든 데이터를 지우려면 앱을 삭제하십시오. 기기에 저장된 노트·음성·모델이 함께 제거됩니다.

8. 아동의 개인정보

본 앱은 누구에게서도 개인정보를 수집하지 않으며, 이는 아동의 경우에도 동일합니다.

9. 방침 변경

본 방침이 변경되면 앱 업데이트 및 본 페이지를 통해 안내합니다.
